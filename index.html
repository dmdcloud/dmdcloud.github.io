Intro

This tool is aimed at developers who want a resource to lookup and search the DMD+D dataset in the UK

DM+D data is publically available, but is difficult to load, and more importantly keep in sync with the constant changes in the dataset. We provide a source of up-to-date data, to be efficiently searched on demand.

We provide a hosted solution, or you can run the system on your own servers (whether real or in the Cloud on AWS & Azure)

Hosted

A great API at a great price:

  • Clear, honest pricing with no hidden costs or lock-in
  • Free developer support - We're around to help you to get started through email or live chat.

On-premise

Our great API on your hardware:

  • Provided as docker appliance
  • Also can be deployed on AWS and Azure
